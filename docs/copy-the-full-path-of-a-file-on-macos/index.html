Copy the full path of a file on macOS

On macOS, we can copy the full path of a file by right-clicking the file, and move to the Copy menu item of the context menu. Press and hold the Alt key, then the menu item's label turns to Copy ... as Pathname.
Clicking the menu item will save the full path to the clipboard.
